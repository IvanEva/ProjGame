
## 📌 Назначение основных папок и файлов

| Путь                      | Назначение                                                                          | Ответственный   |
|---------------------------|-------------------------------------------------------------------------------------|-----------------|
| `main.py`                 | Запуск игры, главный цикл, менеджер сцен                                           | Тимлид          |
| `settings.py`             | Константы (размер экрана, цвета, гравитация, пути к ресурсам)                       | Все программисты|
| `requirements.txt`        | Список зависимостей для `pip install -r requirements.txt`                           | Тимлид          |
| `assets/images/`          | Вся графика, разбитая по категориям (персонажи, тайлы, UI и т.д.)                   | Художник        |
| `assets/sounds/`          | Музыка и звуковые эффекты, сгруппированные по типу                                  | Звукорежиссёр   |
| `assets/levels/`          | Файлы уровней из редактора Tiled (`.tmx`)                                           | Дизайнер уровней|
| `src/entities/`           | Классы игрока (`Player.py`), врагов (`Enemy.py`), снарядов (`Projectile.py`)       | Программист геймплея |
| `src/scenes/`             | Классы сцен: `GameScene`, `MenuScene`, `PauseScene`                                 | Тимлид / Программист |
| `src/ui/`                 | Переиспользуемые элементы интерфейса (`Button`, `HealthBar`)                        | Программист / Художник |
| `src/utils/`              | Вспомогательные функции (загрузка изображений, парсинг уровней, математика)         | Все программисты |
| `docs/`                   | Дизайн-документ, заметки, полезные ссылки                                           | Вся команда     |
Таблица со слоями в файле уровня:

| Слой | Назначение | Тип слоя | Примеры содержимого |
|------|------------|----------|---------------------|
| Background (Дальний фон) | Неподвижный фон, облака, небо | Тайловый или изображение | Градиент неба, горы вдалеке |
| BackgroundDetails | Декорации за игроком, но ближе фона | Тайловый | Дальние деревья, кусты |
| Platforms (Твёрдые платформы) | Всё, по чему можно ходить и с чем сталкиваться | Тайловый | Земля, кирпичи, платформы |
| Objects (Интерактивные объекты) | Объекты, с которыми можно взаимодействовать (не тайлы) | Объектный | Двери, рычаги, сундуки |
| Entities (Сущности) | Точки спавна игрока, врагов, NPC | Объектный | Старт игрока, враги, бонусы |
| ForegroundDetails | Декорации перед игроком (листва, трава) | Тайловый | Трава, цветы, листья деревьев |
| Foreground (Ближний план) | Объекты, которые всегда поверх игрока | Тайловый или объектный | Ветки деревьев, свисающие лианы |
| Collisions (Коллизии, опционально) | Дополнительные области для коллизий | Объектный | Невидимые стены, сложные формы |
