## 🗺️ Таблица слоёв в файле уровня (Tiled)

| Слой                      | Тип слоя        | Назначение                                                                 | Примеры содержимого                          |
|---------------------------|----------------|----------------------------------------------------------------------------|----------------------------------------------|
| `Background`              | Тайловый        | Дальний фон (небо, облака, горы) – не участвует в коллизиях                | Градиент неба, дальние горы                  |
| `BackgroundDetails`       | Тайловый        | Декорации за игроком, но ближе фона                                        | Дальние деревья, кусты                       |
| `Platforms`               | Тайловый        | **Твёрдые платформы** – по ним можно ходить, с ними проверяются коллизии   | Земля, кирпичи, обычные и односторонние платформы |
| `Objects`                 | Объектный       | Интерактивные объекты (не тайлы) – двери, рычаги, сундуки                   | Двери, переключатели                         |
| `Entities`                | Объектный       | Точки спавна игрока (`player`), врагов (`enemy`), бонусов                  | Старт игрока, враги                          |
| `ForegroundDetails`       | Тайловый        | Декорации перед игроком (листва, трава) – рисуются поверх, без коллизий    | Трава, цветы                                 |
| `Foreground`              | Тайловый        | Ближний план – объекты, которые всегда поверх игрока                       | Ветки деревьев, свисающие лианы              |
| `Collisions` (опционально)| Объектный       | Дополнительные области коллизий (невидимые стены, сложные формы)           | Наклонные поверхности, ловушки               |

> **Примечание:** Слои, начинающиеся с `Front` (например, `Front1`–`Front5`), автоматически становятся полупрозрачными при пересечении с игроком (прозрачность задаётся в `settings.py`).

## 🎮 Механики и их реализация

| Механика          | Реализация                                                                 |
|-------------------|----------------------------------------------------------------------------|
| **Движение**      | Класс `Player` обрабатывает клавиши ←/→, изменяет `vel_x` и проверяет коллизии |
| **Бег**           | Зажатие `Shift` увеличивает скорость до `PLAYER_RUN_SPEED` и переключает анимацию на `run` |
| **Прыжок**        | Клавиша `Пробел` вызывает `jump()`, если `on_ground == True`              |
| **Гравитация**    | `vel_y += GRAVITY` каждый кадр, при столкновении снизу `vel_y = 0`         |
| **Односторонние платформы** | Тайлы со свойством `one_way = true` игнорируются при движении снизу вверх |
| **Камера**        | Класс `Camera` смещает отрисовку всех спрайтов, центрируя на игроке        |
| **Анимации**      | Спрайт-листы нарезаются на кадры; скорость анимации задаётся в `settings.py` |
| **Прозрачность передних слоёв** | При пересечении `player.rect` с тайлом из слоя `Front*` весь слой становится полупрозрачным |

## 🛠️ Настройка параметров игры

Все основные параметры вынесены в `settings.py`:

- `LOGICAL_WIDTH`, `LOGICAL_HEIGHT` – разрешение окна (по умолчанию 1920×1080)
- `FULLSCREEN` – полноэкранный режим (`True` / `False`)
- `FPS` – частота кадров
- `PLAYER_SPEED`, `PLAYER_RUN_SPEED`, `PLAYER_JUMP_POWER`, `GRAVITY` – физика
- `PLAYER_FRAME_WIDTH`, `PLAYER_FRAME_HEIGHT` – размер кадра анимации
- `PLAYER_ANIMATIONS_CONFIG` – имена файлов, количество кадров и скорость анимаций
