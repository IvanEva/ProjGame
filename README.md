# Platformer Game — 2D-платформер на Pygame


Простая 2D-игра в жанре платформера с видом сбоку, написанная на Python с использованием библиотеки Pygame.

---

## ✨ Особенности

- Полноэкранный режим с масштабированием (логическое разрешение 800×600)
- Плавное управление персонажем (влево/вправо, прыжок)
- Базовая физика: гравитация, столкновения с блоками
- Поддержка фонового изображения (опционально)
- Чистая архитектура кода, удобная для командной разработки
- Подготовлена структура для ассетов (графика, звуки, уровни)

---

## 🛠 Технологии

- **Python** 3.9–3.12
- **Pygame** 2.5.2
- **Git** для контроля версий

---

## 📋 Требования

- Операционная система: Windows, macOS или Linux
- Установленный Python версии 3.9–3.12 (рекомендуется 3.12)
- Менеджер пакетов `pip`
