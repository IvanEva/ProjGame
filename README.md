# Platformer Game — 2D-платформер на Pygame


Динамичная 2D-игра в жанре платформера с видом сбоку, написанная на Python с использованием библиотеки Pygame. Проект создан для командной разработки и служит основой для дальнейшего расширения: добавления врагов, новых уровней, звуков и графики.

---

## ✨ Особенности

- Полноэкранный режим с масштабированием (логическое разрешение 1920×1080)
- Плавное управление персонажем (влево/вправо, прыжок, бег)
- Анимации: покой, ходьба, бег, атака (загружаются из спрайт-листов)
- Механика бега с увеличенной скоростью (клавиша Shift)
- Физика: гравитация, столкновения с блоками, односторонние платформы
- Камера, следующая за игроком (прокрутка карты)
- Прозрачность передних декоративных слоёв при прохождении персонажа
- Поддержка уровней, созданных в редакторе **Tiled** (формат `.tmx`)
- Чистая архитектура кода, удобная для командной разработки

---

## 🛠 Технологии

- **Python** 3.9–3.12 (рекомендуется 3.12)
- **Pygame** 2.5.2
- **pytmx** 3.32.0 (загрузка карт Tiled)
- **Git** для контроля версий

---

## 📋 Требования

- Операционная система: Windows, macOS или Linux
- Установленный Python версии 3.9–3.12 (рекомендуется 3.12)
- Менеджер пакетов `pip`
